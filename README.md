# AI-Blog
Automating technical study and code archiving through LLM prompt engineering.

### AI를 활용한 자체 공부 블로그 글 작성 시스템 만들기 
- 기존 로컬 환경에서 prompt를 통해 학습하던 방식을 글로 남기기 위해 자동화를 해보려 한다.

# 계획
- 1. 채팅처럼 어떤 학습을 수행할 것인가를 입력 받고 
- 2. 해당 과정을 화면에 나타내고
- 3. 수행과정에서 만들어진 파일을 블로그 글처럼 바로 비동기로 갱신

# 고려 사항 
- 일단 아무래도 내 개인 키를 활용해서 내 서버에 배포하지만 악의적인 누군가가 접근해서 막 사용 가능
- 그렇다고 master 권한 부여해서 특정 사람들만 사용 가능하게 하고 싶지는 않음 
- 만약 어떤 학습을 수행하되 -> 기존에 이미 있는 내용이라면 -> 그냥 바로 해당 markdown 채팅에서 보여주기 
- 프론트, 백은 최대한 경량 구조로 

# 예상 구조
- 채팅 입력 받을 프론트 서버 <- 이건 기존의 랜딩 페이지에 추가 
- 입력 받은 내용 + llm에 요청 및 파일 작성, 수정 권한 갖는 백엔드 서버
